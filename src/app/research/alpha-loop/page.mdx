# AlphaLoop: a multi-agent system for markets that runs itself

Markets are the harshest long-horizon environment we know: noisy feedback,
regime shifts, and a scoreboard that settles every day whether you like it
or not. That's exactly why we build there.

AlphaLoop is a fully automated trading research system for US equities.
Five independent analysis agents — technical, trend, macro, sentiment, and a
proprietary 30-factor quantitative alpha engine — each score every stock in
a ~150-name universe. A fusion layer blends them into a single 0–100
conviction score, adjusted for market regime and each agent's confidence. A
momentum portfolio is constructed from the top names, executed through a
brokerage API behind multiple safety gates, verified after every trade, and
written up in a daily research report that lands in your inbox before the
open.

## Built like infrastructure, measured like a fund

- **One source of truth for strategy math.** The same portfolio-construction
  code drives backtests, daily reports, and live orders — the three can
  never silently drift apart.
- **Honest backtesting.** Next-day execution to kill look-ahead bias,
  transaction costs charged both ways, and per-year sweeps against both the
  S&amp;P 500 and Nasdaq.
- **Live, continuous evaluation.** The system has traded a live paper
  account since March 2026, up 53.8% against the S&amp;P 500's 13.5% over
  the same period — with every trade logged and performance docs generated
  automatically from the broker API.
- **Self-critical by design.** Our own strategy notes decompose returns into
  basket-versus-market and strategy-versus-basket, and flag survivorship
  bias explicitly. We publish the warts on purpose: a system you can trust
  unattended is one that tells you what it can't do.

## Why this matters to us

AlphaLoop is our proving ground for reliable behavioral control: an agent
that acts autonomously with real consequences needs hard guardrails, verified
execution, and an audit trail — the same properties every long-horizon agent
will need. (It's a research system, not investment advice.)
